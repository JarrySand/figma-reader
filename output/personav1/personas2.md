### ⚠️ スクリーニング基準（受け入れ/非受け入れの線引き）

このペルソナは**最も取り扱いに注意が必要**。DAO大学の理念に共感する層と、エコシステムを毀損するリスクのある層が混在する。

| 受け入れ可能な特徴 | 受け入れに注意が必要な特徴 |
|-------------------|-------------------------|
| ・理想を持ちつつ「具体的に何をすべきか」を学びたい姿勢がある | ・特定の思想・教義を広めることが主目的 |
| ・他者との対話・協働に開かれている | ・排他的・攻撃的な言動がある |
| ・自分の考えを疑い、アップデートする柔軟性がある | ・科学的根拠を否定する主張を繰り返す |
| ・実行（手を動かすこと）への意欲がある | ・他者を勧誘する目的で参加しようとする |
| ・DAO大学の行動規範に同意できる | ・DAO大学のプラットフォームを自己の布教に利用する意図がある |

### スクリーニングの仕組み

1. **申込時アンケート**: 参加動機・期待する成果・行動規範への同意を確認
2. **カジュアル面談**: 1対1の30分面談で対話姿勢を確認
3. **トライアル期間**: 最初の1ヶ月はトライアル参加として、コミュニティとの適合を双方で確認
4. **行動規範の明文化**: 「多様な意見を尊重する」「特定の思想の勧誘を行わない」「科学的対話を重んじる」
5. **メンターの配置**: D-3タイプには特にメンター（A-3山本のような人物）を手厚く配置

### ⚠️ フリーライド対策（D-4タイプへの設計指針）

このペルソナは「負担者」だが、**動機が利己的リターンに偏る場合、エコシステムの価値観を毀損するリスク**がある。

#### リスクと対策

| リスク | 対策 |
|--------|------|
| リターンだけ受け取りコミュニティに関与しない | トークン/ポイントの一部を「参加・貢献」に連動させる設計 |
| 投機的動機で参加し、トークン価格を煽る | トークンの投機的売買を制限する設計（ロック期間・用途限定） |
| 「社会貢献」を税務上のテクニックとしてのみ利用 | 寄付の使途報告を義務化し、インパクトの可視化で本質的関心を引き出す |
| 高額寄付者としてガバナンスに過度な影響力を持とうとする | 寄付額とガバナンス投票権を非比例にする設計（1人1票 or 上限付き加重） |

#### D-4向けのジャーニー設計方針

| 段階 | 設計 |
|------|------|
| **認知** | ふるさと納税ポータル・投資メディアでリーチ（入口はリターン訴求でOK） |
| **興味** | 節税シミュレーション＋活動報告で「意味のある使い方」を提示 |
| **参加** | ふるさと納税 or サポーター会員としてまず小額から参加 |
| **体験** | 活動報告書・受講生からの感謝メッセージ → 「自分のお金がこう使われている」の実感 |
| **変容** | 「節税テクニック」→「応援したいから出す」への動機の進化 |
| **伝播** | 投資コミュニティで「リターン以上の満足感がある」と発信 |
| **理想的な役割移行** | **リターン目的の負担者 → 意義を感じる支援者 → 自らも受講・参加（受益者兼負担者へ）** |
