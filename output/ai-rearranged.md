# 日本DAO協会（DAO大学） ステークホルダーマップ — AI Rearranged

> FigJamボード「FigJamへようこそ」のAI整理済みセクションを網羅的にまとめたもの

---

## 構造概要

```
提供する人（負担者）  ──講師、コンテンツ──→  日本DAO協会（DAO大学）  ──定款、授業、イベント──→  受ける人（受益者）
        ↑                                    │                                        │
        │                              蓄積アセット:                                  │
        │                         ノウハウ（アセット）                                 │
        │                         ネットワーク（ステークホルダー）                     │
        │                         ブランド（トークン/ポイント）                        │
        │                                                                              ↓
        └──────────── DAO広める / 参加する / お金払う ──────────────────────────────────┘
```

ステークホルダーの役割: **①応援する人　②やる人　③受け取る人**

受益者への提供: **講座助成する / 参加者を受け入れる**

提供する人への提供: **リソース**
