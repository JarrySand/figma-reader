# DAO リソース設計書 — ヒト（労働力・専門性）の逆算設計

> **目的：** DAOに労働力・専門性・時間を提供する可能性がある人をプールし、その人たちが大量に投入されるためのキーマンを明確にする
>
> **設計手順：**
> ① FigJamターゲット群から「ヒト＝労働力や専門性を提供する」可能性がある人をリスト化
> ② その人たちが多く投入されるためのキーマンをまとめる

---

## ヒト提供者の全体規模感

```
                        1人あたりの投入時間/週
                              ↑
    フルタイム（30h+）        │  H1-2定年退職者  H1-4退任自衛官
                              │  H2-6シェアハウス  H3-2DAOで食いたい人
                              │  H6-6地域おこし協力隊

    ハーフタイム（10-30h）    │  H1-1経験活かしたい人  H1-3退職直後
                              │  H1-5子供卒業した母  H2-1〜H2-3学生
                              │  H4-3外国人ノマド  H6-1大学インターン
                              │  H6-4企業派遣社員
                              │
    パートタイム（5-10h）     │  H1-6市民活動  H1-7社会理論
                              │  H3-3ブロックチェーン  H3-5スタートアップ
                              │  H3-6エグジット組  H5-5哲学者
                              │  H5-7文化保全  H7-5既存事業者

                              │  H5-1〜H5-4発信系
                              │  H5-6祭り  H3-4コンサル
                              │
                              └──────────────────────────→ 人数の多さ
                                少数・高専門性        大量・汎用
```

## H1群（スキル持て余し系）のキーマン

| キーマン | なぜキーマンか | 動かし方 | 動いた時の効果 |
|---------|-------------|---------|-------------|
| **シルバー人材センターの所長** | H1-1〜H1-3（退職者・定年者）は「次の活動先」をシルバー人材センターで探す人が多い。所長が「こんな活動もある」と紹介すれば一気に流入 | ①シルバー人材センターとの連携協定 ②「メンター募集」の掲示 ③所長への説明会 | 退職シニアが安定的に供給される。週10h以上の高稼働人材が月数名ペースで |
| **自衛隊地方協力本部（地本）の再就職担当** | H1-4（退任自衛官）の再就職はほぼ地本経由。担当者が紹介先リストにDAO大学を入れれば候補者が来る | ①地本への説明訪問 ②「危機管理・ロジスティクス」ポジションの提示 ③退官者向け説明会での登壇 | ロジスティクス・リーダーシップの即戦力。年間数名だが質が極めて高い |
| **PTA連合会 / 地域の子育てネットワークのリーダー** | H1-5（子供卒業した母）はPTAのネットワークに残っていることが多い。リーダーの紹介で動く | ①PTA連合会での活動紹介 ②「OGメンバー向け新しい活動の場」としての訴求 ③口コミで広がる仕掛け | コミュニティ運営・事務処理の実務者が安定供給。特に地域密着型の運営で力を発揮 |
| **NPO支援センター / ボランティアセンターの所長** | H1-6（市民活動経験者）は活動終了後にNPO支援センターに相談する。所長が「次の活動先」として紹介 | ①NPO支援センターとの連携 ②「活動の場を探している人」向けの情報提供 ③センター主催のマッチングイベントに参加 | NPO運営ノウハウを持つ人材の獲得。助成金申請・行政折衝の経験者は特に貴重 |
